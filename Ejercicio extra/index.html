  • Novedades en HTML 5

    • Una de las principales novedades que trae HTML5 es un conjunto de nuevas etiquetas semánticas que permiten estructurar mejor la creación de las páginas web, algunas de ellas como <header>, <footer>, <article>; para organizar mejor el contenido en nuestra página.

    • Se introdujo un soporte nativo de multimedia para poder incluir archivos de audio y vídeo directamente sin necesidad de utilizar herramientas externas mediante las etiquetas <audio> y <video>.

    • También incluye una característica de responsive design para adaptar los sitios web a diferentes tamaños de pantalla.

    • Una API de almacenamiento local que permite almacenar datos en el dispositivo del usuario para poder seguir funcionando aún sin conexión.

    • Opciones de geolocalización que detectan automáticamente la ubicación de quien accede a la página permitiendo, por ejemplo, reenlazar al usuario al sitio específico de un país en dónde se encuentre.

  • Web Storage

    Es una API que permite guardar grandes cantidades de datos localmente en el navegador del usuario sin afectar al rendimiento del sitio web. Provee de dos tipos de objetos para ello:

    1. window.localStorage: Guarda datos sin fecha de expiración, se mantienen aún después cerrar el navegador.

    2. window.sessionStorage: Funciona de manera similar al objeto localStorage con la diferencia que al culminar la sesión (cerrar el navegador) se eliminan los datos.

  • Elemento <iFrame>

    Ésta etiqueta de HTML nos permite insertar un documento html dentro de otro, a modo de incluir una ventana a otro sitio web dentro de mi sitio web, se necsita proveer de una URL para brindar la ubicación del documento a insertar.
